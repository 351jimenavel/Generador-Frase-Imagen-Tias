✨ Frases de Tía con IA 🤍
Una aplicación web divertida y nostálgica que permite crear frases motivacionales al estilo "tía", acompañadas de imágenes generadas por inteligencia artificial usando la API de Novita.


🧠 Funcionalidades:
- Escribí tu frase personalizada.
- Elegí un estilo visual: Piolín 🐥, Flores 🌸, Animales 🐱, o Dios ✝️.
- Se conecta al backend para generar una imagen usando la API de Novita.
- Vista previa de la imagen generada.
- Soporte para modo oscuro 🌓.

🛠 Tecnologías utilizadas:
Frontend: HTML5, CSS3, JavaScript
Backend: Python + Flask
API: Novita.ai

📁 Estructura del proyecto:
- app.py
- .env
- templates/index.html
- static/script.js
- README.txt

🔧 Instalación:
1. Cloná el proyecto
2. Instalá dependencias: pip install flask python-dotenv requests
3. Crea un archivo .env con tu clave:
   NOVITA_API_KEY=sk_tu_clave_aqui
4. Ejecutá el servidor: flask run
5. Navegá a http://localhost:5000



🎯 Estilos disponibles:
- piolin
- flores
- animales
- dios

❤️ Hecho con cariño para todas tus frases especiales.
